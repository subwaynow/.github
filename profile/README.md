지금지하철은 서울 지하철의 실시간 도착 정보, 열차 위치,
그리고 지금 이 순간 가장 필요한 지하철 정보를
간단하고 직관적으로 제공하는 사용자 중심 서비스입니다.

📍 역별 실시간 열차 도착 정보

⏱ 지연·혼잡 등 현재 상태 중심 안내

🗺 노선 기반 빠른 탐색
